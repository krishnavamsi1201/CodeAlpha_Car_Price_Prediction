# 🚗 Car Price Prediction using Machine Learning

## 📊 Features
- Data preprocessing
- Feature encoding
- Train-test split
- Linear Regression model
- Accuracy evaluation
- Data visualization graphs

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
